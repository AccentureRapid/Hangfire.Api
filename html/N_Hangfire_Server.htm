Hangfire.Server Namespace
The Hangfire.Server namespace contains types that are responsible for background processing. You may use them to customize your processing pipeline by implementing the IServerFilter interface or define your own continuously-running background processes by implementing the IBackgroundProcess as well as create completely custom instances of BackgroundServer.
